# Checklist de accesibilidad — Convergencia Serena

## Global

- [ ] `main`, `header`, `nav`, `aside`, `footer` presentes.
- [ ] Skip link visible al foco.
- [ ] Orden de tabulación coincide con orden visual.
- [ ] No hay trampas de teclado.
- [ ] Foco visible con outline ≥ 3px.
- [ ] Targets ≥ 44×44 px.
- [ ] Texto normal contraste ≥ 4.5:1.
- [ ] Texto grande contraste ≥ 3:1.
- [ ] Estados no dependen solo del color.
- [ ] `prefers-reduced-motion` respetado.

## Builder ARASAAC

- [ ] Búsqueda manual etiquetada.
- [ ] Resultados con `alt` del pictograma real.
- [ ] Botones describen la acción exacta.
- [ ] Atribución siempre visible.
- [ ] Estado de revisión en `aria-live`.
- [ ] Exportación deshabilitada hasta aprobación.

## Dark theme

- [ ] No usa inversión automática.
- [ ] Bordes y superficies distinguibles.
- [ ] Foco visible en cyan/blue.
- [ ] Warning/disabled legibles.
