# Frontend Design System — Convergencia Serena

Los tokens de `tokens.css` son la fuente canónica de color, espaciado, radios y
elevación del frontend.

## Reglas

- Mantener contraste WCAG 2.2 AA.
- No depender solo del color para comunicar estados.
- Usar HTML semántico, targets mínimos de 44 px y foco visible.
- No generar, imitar ni modificar pictogramas ARASAAC.
- Mantener revisión humana y atribución visibles.
- No añadir recursos de red para tipografías, iconos o decoración.
