# Dossier de validación institucional ARASAAC

## Resumen ejecutivo

ARASAAC Social MCP Platform es una plataforma **no comercial** para preparar materiales accesibles con pictogramas **reales** de ARASAAC, revisión humana obligatoria y atribución visible CC BY-NC-SA.

## Cumplimiento licencia

- Solo pictogramas obtenidos desde API/host oficial `arasaac.org`.
- Sin generación, imitación ni modificación de pictogramas con IA.
- Atribución visible en UI y en cada exportación.
- Autor: Sergio Palao. Propietario: Gobierno de Aragón.

## Gobernanza operativa

- Revisión humana antes de exportar.
- Auditoría append-only de creación, revisión y exportación.
- Sin datos personales sensibles en MVP.
- Sin vinculación a personas concretas.

## Stack y despliegue

- Backend FastAPI, frontend Next.js AA, PostgreSQL, Docker Compose.
- MCP stdio con allowlist de tres tools de pictogramas.

## Solicitud de validación

Se solicita revisión institucional de ARASAAC / Gobierno de Aragón para uso piloto en entidades sociales, CEE y fundaciones, manteniendo licencia CC BY-NC-SA y control humano.
