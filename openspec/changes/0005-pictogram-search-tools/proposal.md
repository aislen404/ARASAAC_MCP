# Proposal — 0005-pictogram-search-tools

## Objetivo

Exponer búsqueda y consulta de pictogramas reales mediante REST y MCP con schemas
estrictos, selección humana y trazabilidad completa.

## Alcance

`search_pictograms` y `get_pictogram`. `suggest_pictograms_for_text` se limita a
dividir texto genérico y buscar términos; no usa IA generativa ni realiza
diagnóstico.
