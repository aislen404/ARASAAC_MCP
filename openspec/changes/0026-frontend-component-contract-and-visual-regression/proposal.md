# Proposal — 0026 Frontend Component Contract and Visual Regression

## Problema

La implementación Convergencia Serena previa cubre funciones del MVP, pero no reproduce la dirección visual aprobada. El resultado actual muestra una jerarquía de formulario, poca identidad, ausencia de assets ricos, carencia de composición tipo producto y una traducción insuficiente a dark theme.

## Cambio propuesto

Crear una capa frontend Convergencia Serena con:

- contratos por componente;
- tokens light/dark completos;
- assets SVG propios;
- layout responsive especificado;
- paneles visuales de producto;
- adaptación del constructor funcional al shell de producto;
- pruebas de regresión visual y accesibilidad;
- rúbrica de aceptación objetiva.

## Fuera de alcance

- Cambiar endpoints backend.
- Cambiar gobernanza ARASAAC.
- Generar pictogramas por IA.
- Introducir autenticación.
- Añadir funcionalidades de red social no existentes.

## Valor

Convierte el MVP funcional en una experiencia reconocible, demostrable, accesible y alineada con el concepto visual aprobado.
