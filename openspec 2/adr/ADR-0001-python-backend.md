# ADR-0001 — Python backend

## Decisión

Usar Python/FastAPI como backend principal y MCP server, alineado con la preferencia del proyecto y facilidad para IA, validación, exportación y servicios.

## Consecuencias

- Menor fricción para servicios IA.
- Pydantic como contrato fuerte.
- Ecosistema amplio para exportación y datos.
