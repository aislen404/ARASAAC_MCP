# Workflow: cermi_foundation_kit

## Pasos

1. **select_goal** — Seleccionar objetivo del kit
2. **easy_reading** — Adaptar a lectura fácil
3. **cognitive_review** — Revisión de accesibilidad cognitiva
4. **search_pictograms** — Buscar pictogramas reales ARASAAC
5. **edit** — Edición por usuario/profesional
6. **institutional_review** — Revisión institucional
7. **export** — Exportar con créditos y manifiesto

## Reglas transversales

- Solo pictogramas reales ARASAAC.
- Revisión humana obligatoria antes de exportar.
- Atribución visible en toda exportación.
- Sin datos personales en MVP.
- OpenSpec aprobada para cambios de producto/código.
